# AI 工具导航平台 - 项目章程

## 项目基本信息

| 项目 | 内容 |
|------|------|
| **项目名称** | AI 工具导航平台 (AI Tool Navigator) |
| **项目目标** | 为 AI PM 面试打造一个完整的面试作品集项目 |
| **项目经理** | AI 高级项目经理（AI辅助） |
| **开发人员** | 你 + AI 辅助 |
| **时间预算** | 每天 1-2 小时，无硬性截止日期 |
| **技术栈** | Vue 3 + Python FastAPI + SQLite + Tailwind CSS |
| **文档语言** | 中文 |

## 项目目标

1. **交付可运行的产品**：包含浏览、搜索、提交工具等功能
2. **交付完整的项目管理文档**：PRD、决策日志、风险登记表、复盘报告
3. **准备面试展示材料**：5 分钟演示脚本、Q&A 准备

## 项目范围

### 在范围内
- 首页（热门推荐 + 分类入口）
- 分类浏览页面
- 工具详情页面
- 实时搜索功能
- 提交工具表单
- 关于页面
- 完整的项目管理文档

### 不在范围内
- 用户注册/登录
- 评分/评论系统
- 管理后台
- 多语言支持
- 数据分析/统计

## 项目阶段

| 阶段 | 内容 | 预估时间 |
|------|------|---------|
| 第一阶段 | 项目骨架搭建（环境、数据库） | 2-3 天 |
| 第二阶段 | 后端 API 开发 | 3-5 天 |
| 第三阶段 | 前端页面开发 | 5-8 天 |
| 第四阶段 | 响应式打磨 + 错误处理 | 2-3 天 |
| 第五阶段 | 面试材料准备 + 部署 | 2-3 天 |

## 验收标准

1. 网站可在本地浏览器正常访问
2. 所有 P0 功能（首页、分类、搜索、详情）可正常使用
3. 提交工具表单可正常提交
4. 手机端和桌面端均可正常显示
5. 项目文档完整
