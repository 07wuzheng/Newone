# 决策日志 - AI工具导航平台

记录项目中的关键决策及理由，面试时可展示决策思维。

---

## ADR-001：项目选择

| 项目 | 内容 |
|------|------|
| **日期** | 2026-06-03 |
| **决策** | 开发 AI 工具导航平台 |
| **备选** | ① AI工具导航平台 ② 项目管理工具 ③ 个人博客CMS |
| **选定** | ① |
| **理由** | 贴合 AI PM 岗位方向；技术难度适中；主题本身展示 AI 行业理解 |
| **权衡** | 博客CMS有更成熟的需求模式，但不如导航平台贴合岗位 |

## ADR-002：前端框架

| 项目 | 内容 |
|------|------|
| **日期** | 2026-06-03 |
| **决策** | Vue 3（Composition API） |
| **备选** | ① Vue 3 ② React ③ 纯HTML/JS |
| **选定** | ① |
| **理由** | 之前学过 Vue，捡起门槛低；Vue 3 性能好；Composition API 代码组织清晰；国内面试 Vue 占比高 |
| **权衡** | React 生态更庞大，但 Vue 上手更快 |

## ADR-003：后端框架

| 项目 | 内容 |
|------|------|
| **日期** | 2026-06-03 |
| **决策** | Python FastAPI |
| **备选** | ① FastAPI ② Flask ③ Node.js Express |
| **选定** | ① |
| **理由** | 学过 Python；自动生成 API 文档（/docs）；异步支持好；类型检查减少 bug |
| **权衡** | Flask 更轻量，但 FastAPI 开发效率和文档能力更强 |

## ADR-004：数据库

| 项目 | 内容 |
|------|------|
| **日期** | 2026-06-03 |
| **决策** | SQLite（通过 SQLAlchemy） |
| **备选** | ① SQLite ② MySQL ③ PostgreSQL |
| **选定** | ① |
| **理由** | 零配置，单文件数据库，适合个人项目；SQLAlchemy 方便后续切换 |
| **权衡** | 不支持并发写入，但个人项目完全不需要 |

## ADR-005：样式框架

| 项目 | 内容 |
|------|------|
| **日期** | 2026-06-03 |
| **决策** | Tailwind CSS |
| **备选** | ① Tailwind CSS ② Bootstrap ③ 手写CSS |
| **选定** | ① |
| **理由** | 快速原型开发；响应式内置；可定制；Vue 社区主流选择 |
| **权衡** | HTML 会稍显冗长，但开发效率最高 |

## ADR-006：MVP 范围定义

| 项目 | 内容 |
|------|------|
| **日期** | 2026-06-03 |
| **决策** | 仅包含浏览、搜索、提交基础功能 |
| **排除** | 用户注册登录、评分系统、管理后台 |
| **理由** | 核心价值是"发现 AI 工具"，浏览搜索即核心；认证和后台增加复杂度但不提升面试价值 |
| **权衡** | 产品看起来不像"完整平台"，但作为面试作品，功能聚焦反而更有利 |

## ADR-007：图片来源

| 项目 | 内容 |
|------|------|
| **日期** | 2026-06-03 |
| **决策** | 使用 picsum.photos 占位图 |
| **备选** | ① picsum.photos ② Pexels ③ Unsplash |
| **选定** | ① |
| **理由** | Pexels 已知有 403 错误问题；Unsplash 需要 API key；picsum.photos 最稳定，无需认证 |
| **权衡** | 图片随机无主题，但作为占位图完全够用 |
