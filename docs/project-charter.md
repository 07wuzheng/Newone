# Project Charter / 项目章程
# AI Tool Navigator - AI工具导航平台

---

## 1. Project Purpose / 项目目的

**EN**: Create a portfolio-quality project demonstrating full-cycle product management and AI-assisted development skills for an AI Project Manager job application.

**CN**: 打造一个能展示全周期产品管理和AI辅助开发能力的面试作品，用于应聘AI项目经理岗位。

## 2. Project Manager / 项目经理
- 角色承担者：你（面试候选人）
- AI 辅助：SeniorProjectManager (WorkBuddy助手)
- 开发方式：AI辅助开发，你负责决策和验收

## 3. Project Scope / 项目范围

**In Scope (MVP)**:
- Homepage with featured tools
- Category browsing (5-6 categories, 4-5 tools each)
- Tool detail pages
- Search functionality
- Submit tool form
- About page
- Responsive design

**Out of Scope**:
- User authentication
- Real reviews/ratings
- Analytics dashboard
- Admin panel
- Social features

## 4. Key Deliverables / 关键交付物

| Deliverable | Format | Owner |
|------------|--------|-------|
| PRD | Document | PM (You) |
| Working Web App | Vue + FastAPI | AI-assisted dev |
| Project Task List | Document | PM (You) |
| Risk Register | Document | PM (You) |
| Retrospective Report | Document | PM (You) |
| Interview Demo Script | Document | PM (You) |

## 5. Stakeholders / 干系人
- **You**: Project Owner + Developer + Interview Candidate
- **SeniorProjectManager (WorkBuddy)**: PM Advisor + Code Generator
- **Future Interviewers**: End users of the portfolio

## 6. Success Criteria / 成功标准

1. MVP features all working and demo-able ✓
2. 5+ PM artifacts produced ✓
3. Can present in 5-minute interview walkthrough ✓
4. Can answer PM-related interview questions using project examples ✓
